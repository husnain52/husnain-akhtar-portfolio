About Me

I am Husnain Akhtar from Rawalpindi, Pakistan. I am a Front end Web Developer with my specialties in React JS, Nexts JS, Material UI, Microsoft Fluent UI, CSS, Node Sass, NPM, jQuery, HTML, Bootstrap & UI/UX.

I am qualified with DAEIT course from CTTI Islamabad after which I started learning Developing Basics from a software house. Then I migrated to React JS a library that inspired most. From that day I started working with React JS and now I am capable of building any kind of UI design, Landing pages and Admin Dashboard Login & Signup. I have over 2+ years of experience in development.
